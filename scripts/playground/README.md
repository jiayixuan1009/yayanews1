# 🛠️ Scripts Playground

这个文件夹最初是在项目根目录（`/`）中建立的各种一次性 Python、Node.js 爬虫探测与 API 测试脚本（例如 `check_api.py`, `query_topics.py` 等）。

为了保证 Monorepo 环境的整洁，并且不污染正式版本的容器镜像边界，它们统统被集中迁移并隔离到了此处！

> **⚠️ 注意事项：**
> 1. 本文件夹路径已经被 `.gitignore` 包含（针对其内部生成的 `*_out.txt`, `*.json`, `*.html` 测试日志）。
> 2. 请以本目录为工作区自由写测试脚本，不要在外部的 `apps/web` 或是 `/根目录` 新建临时测试文件，避免提交到线上！
> 3. 运行前请确保能正确导入并读取上层 `.env`。
